Solidity поддерживает неявные типы: тип i в var i = 42; — это наименьший целочисленный тип, достаточный для хранения значения в правой части выражения (uint8). Рассмотрим общий шаблон для цикла: 


for (var i = 0; i < array.length; i++) { /* ... */ }

Тип i приводится к uint8. Если array.length больше 255, произойдет переполнение. Явно определяйте тип при объявлении целочисленных переменных: 


for (uint256 i = 0; i < array.length; i++) { /* ... */ }

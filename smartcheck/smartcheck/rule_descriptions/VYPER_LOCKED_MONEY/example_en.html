In the following example the contract implements a function with @payable decorator: 


funders: {sender: address, value: wei_value}[int128]
nextFunderIndex: int128


@public
@payable
def participate(i: int128):
    nfi: int128 = self.nextFunderIndex
    self.funders[nfi] = {sender: msg.sender, value: msg.value}
    self.nextFunderIndex = nfi + 1

However, it is impossible to withdraw ETH from the contract address.

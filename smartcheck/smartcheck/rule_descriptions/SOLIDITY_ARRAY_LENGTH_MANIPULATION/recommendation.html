По возможности не работайте с длиной динамического массива напрямую.

  • Используйте uint[] storage arrayName = new uint[](7) для создания динамического массива нужной длины.
  • Используйте delete arrayName для очистки динамического массива.
  • Используйте .push() (вместо .length++) для записи в конец массива.
  • Начиная с версии компилятора 0.5.0, спользуйте .pop() (вместо .length--) для удаления последнего элемента массива.

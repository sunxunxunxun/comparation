In the following example, checking for strict balance equality is used: 


@public
def func_1(user: address):
    if (user.balance == as_wei_value(10, "ether")):
        # do some stuff
        pass

Secure alternative: 


@public
def func_1(user: address):
    if (user.balance >= as_wei_value(10, "ether")):
        # do some stuff
        pass

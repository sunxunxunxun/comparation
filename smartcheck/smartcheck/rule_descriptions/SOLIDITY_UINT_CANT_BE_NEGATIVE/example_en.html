The following example includes an infinite loop: 


pragma solidity 0.4.24;

contract GreaterOrEqualToZero {

    function infiniteLoop(uint border) returns(uint ans) {

        for (uint i = border; i >= 0; i--) {
            ans += i;
        }
    }
}

In this case, i >= 0 condition will always evaluate to true. The next value of i variable after 0 will be 2**256-1. Thus, the loop will be infinite.

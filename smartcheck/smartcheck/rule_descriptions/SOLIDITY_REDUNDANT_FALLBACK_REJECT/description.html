Fallback-функция по отклонению платежа является избыточной.

Контракты должны отклонять неожиданные платежи. До версии 0.4.0 в Solidity это делалось вручную: 


function () { revert(); }

Начиная с Solidity 0.4.0, контракты без fallback-функции автоматически возвращают платежи, что делает код, представленный выше, избыточным.

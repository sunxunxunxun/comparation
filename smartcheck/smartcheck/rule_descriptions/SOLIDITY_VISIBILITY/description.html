В Solidity есть уровни видимости по умолчанию. Для функций в контрактах - public, для функций в интерфейсах - external, для переменных - internal. Fallback функция в контрактах может иметь только external или public уровень видимости. Все функции в интерфейсах должны иметь external уровень видимости. Четко определяйте видимость для предотвращения неоднозначной ситуации.
